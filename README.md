# Sistema de Tarefas em Java (ToDo List)

Projeto desenvolvido em Java com o objetivo de praticar conceitos de programação orientada a objetos, manipulação de listas e persistência de dados em arquivo.

## Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Marcar tarefa como concluída
- Remover tarefa
- Salvamento automático em arquivo
- Carregamento automático ao iniciar o sistema

## Tecnologias utilizadas

- Java
- IntelliJ IDEA
- Git e GitHub

## Como executar

1. Clone o repositório

git clone https://github.com/leandro-2025/sistema-tarefas-java

2. Abra o projeto no IntelliJ

3. Execute a classe `Main.java`

## Exemplo de uso

1 | [ ] | Estudar Java  
2 | [✔] | Fazer exercícios  

## Objetivo do projeto

Este projeto foi criado para praticar:

- Programação orientada a objetos
- Estrutura de dados (ArrayList)
- Manipulação de arquivos (FileWriter e BufferedReader)
- Organização de código em camadas (model, service, main)
