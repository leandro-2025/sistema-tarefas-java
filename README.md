📝 Sistema de Tarefas em Java

📌 Descrição
Sistema de gerenciamento de tarefas desenvolvido em Java com persistência em banco de dados SQLite.
Permite adicionar, listar, editar, concluir e remover tarefas.

🚀 Funcionalidades
✅ Adicionar tarefa
📋 Listar tarefas
✔️ Concluir tarefa
✏️ Editar tarefa
❌ Remover tarefa
🛠️ Tecnologias utilizadas
Java
SQLite
JDBC

▶️ Como executar o projeto
Clone o repositório:
git clone https://github.com/leandro-2025/sistema-tarefas-java/edit/main/README.md
Abra no IntelliJ IDEA
Execute a classe Main

💡 Aprendizados
Neste projeto pratiquei:
CRUD com banco de dados
Uso de PreparedStatement
Integração Java com SQLite
Boas práticas de organização de código
